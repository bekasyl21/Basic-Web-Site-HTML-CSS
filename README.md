# 🍩 LA Donut — Los Angeles Donut Culture Website

A maximalist editorial website about Los Angeles donut culture, built in pure HTML/CSS/JS. All content is in **Kazakh language**.

## Files

| File | Description |
|------|-------------|
| `index.html` | Homepage — hero, stats, featured donuts, places preview |
| `menu.html` | Menu page — full donut catalog with photos, filters, combos |
| `about.html` | About page — history timeline, legendary spots, Cambodian heritage |
| `style.css` | Shared styles — fonts, nav, footer, animations, utilities |

## Stack

- Vanilla HTML5 / CSS3 / JavaScript
- Google Fonts: Playfair Display, Unbounded, Mulish
- Photos: Unsplash (no API key needed)
- No frameworks, no dependencies

## Design

- **Palette:** warm cream `#FFFDF8`, brown `#3D1A00`, pink `#F2345A`, gold `#D4A017`
- **Style:** maximalist editorial — magazine layouts, full-bleed photos, bold typography
- **Features:** custom cursor, scroll reveal animations, sticky nav, running marquee

## Usage

Download all 4 files into the same folder and open `index.html` in a browser. No build step required.

```
la-donut/
├── index.html
├── menu.html
├── about.html
└── style.css
```

## Content

Covers LA donut history from 1952, Cambodian immigrant culture (≈70% of LA donut shops), and iconic locations: Randy's Donuts, The Donut Man, Fonuts, DK's Donuts.
